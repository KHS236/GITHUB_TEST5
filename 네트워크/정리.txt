3. LAN 의 특징에 대해서 서술하세요

F - 근거리 통신망 / 고속 통신 목적

비교적 작은 지리적 범위에서 사용하는 네트워크

주요 특징
- 지리적 범위 : 제한된 지역 내 사용
- 속도 : 고속 전송 (100MB~10GBPS)
- 비용 : 낮은 설치/운영 비용
관리 용이성 : 간단한 설치와 유지보수
보안성 : 외부보다 보안이 쉽다

구성 요소
- NIC : 유/무선 네트워크 연결
- Switch : 데이터 패킷 목적지로 전달
- Router : 외부망과 연결, DHCP, 보안기능 제공
- WAP : Wi-Fi 연결 지원
- Cables : 장치간 물리적 연결
- Hub : 브로드캐스트 방식 연결 (구형 장비)
- Server : 파일, 웹, DB 서비스제공


4. WAN의 특징에 대해서 서술하세요

F - 광대역 / 원거리 통신망 , 장치간 거리가 아주 먼 형태 , 다양한 데이터 Type 교환, 안정성(신뢰도) 중점

도시, 국가, 대륙 등 넓은 범위를 연결하는 통신망

주요 특징
- 광범위한 커버리지
- 다양한 전송 기술 사용
- 인터넷 기반 연결 가능 (VPN 포함)
- 높은 운영 비용
- 복잡한 네트워크 관리 필요

구성 요소
- Router : LAN과 WAN연결 라우팅 수행
- Transmission Media : 유선(광, 전용선), 무선(위성,LTE 등)
- Switch : 일부 환경에서 사용된다.
- Modem : 아날로그 <-> 디지털 신호 변환
- CSU/DSU : 전용 회선 연결 장비

8. TCP/IP 모델과 OSI 7 모델의 계층을 적고 각 계층별 역할을 적으세요


1계층(물리)     :   장치간 물리적 연결
2계층(데이터)   :   장치간(Point to Point) 데이터의 운송 방식 결정
3계층(네트워크) :   경로 탐색
4계층(전송)     :   송수신지간(End to End) 데이터의 운송 방식 결정
5계층(세션)     :   지속적인 데이터 교환을 위한 대화형 채널 생성
6계층(표현)     :   jpg , 작업물 압축 , 인코딩 , 암호화 등 7계층(응용)의 작업물에 대한 저장 방식
7계층(응용)     :   통신을 수반하는 프로그램(사용자/장치에 의해 작업물 생성)
